== Description ==

You can choose color of admin's comments

== Installation ==

for wordpress 2.6.5 :
download version 1.0.0
Upload the admin-comment to your blog, Activate it.
put chmod 777 to the plugin's directory.
go to admin panel and choose your color.
This plugin don't work with all themes.

For wordpress 2.7 :
Download version 2.2.0
Upload the admin-comment to your blog, Activate it.
